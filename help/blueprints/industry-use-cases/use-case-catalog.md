# Catálogo de casos de uso

Casos de uso do setor mostram como as organizações em mercados verticais específicos aplicam o Adobe Experience Platform e os aplicativos para alcançar resultados mensuráveis nos negócios. Cada caso de uso descreve um cenário comercial concreto, seu impacto esperado e links para o [padrão de caso de uso](/help/blueprints/use-case-patterns/overview.md) que fornece orientação detalhada de implementação.

Navegue por setor para encontrar casos de uso relevantes para sua organização e, em seguida, siga os links padrão para referências de implementação, incluindo orientação de decisão, cadeias de função e documentação do Experience League.

| Setor | Temas principais |
| --- | --- |
| [Automotivo](automotive/automotive-overview.md) | Jornada de compra de veículo, ciclo de vida do serviço, experiências de carro conectadas, fidelidade do proprietário |
| [B2B](b2b/b2b-overview.md) | Marketing baseado em conta, pontuação de leads, aceleração do pipeline, expansão de clientes |
| [Serviços financeiros](financial-services/financial-services-overview.md) | Recomendações de produto, prevenção de churn, ofertas em estágio de vida, personalização de fraudes |
| [Assistência médica](healthcare/healthcare-overview.md) | Gestão de consultas, adesão a medicamentos, integração de pacientes, coordenação de cuidados |
| [Seguros](insurance/insurance-overview.md) | Renovação de políticas, experiência em sinistros, prevenção de riscos, otimização de venda cruzada |
| [Mídia e entretenimento](media-entertainment/media-entertainment-overview.md) | Recomendações de conteúdo, retenção de assinatura, conversão de avaliação, envolvimento entre plataformas |
| [Varejo](retail/retail-overview.md) | Personalização de produto, recuperação de carrinho, otimização de venda cruzada, envolvimento de fidelidade |
| [Telecomunicações](telecommunications/telecommunications-overview.md) | Atualizações de dispositivos, prevenção de churn, otimização de planos, participação na rede |
| [Viagens e hospitalidade](travel-hospitality/travel-hospitality-overview.md) | Personalização de reserva, recuperação de abandono, programas de fidelidade, campanhas sazonais |
| [Tecnologia](technology/technology-overview.md) | Coleta de eventos, encaminhamento de dados em tempo real, integração de análises, implantação de borda |

## Como os casos de uso se conectam às orientações de implementação

Cada caso de uso está vinculado a um **padrão de caso de uso** — uma abordagem de implementação repetível que descreve a cadeia de função, os pontos de decisão e as etapas de configuração necessárias para dar vida ao caso de uso. Por sua vez, os padrões de casos de uso conectam-se a [objetivos comerciais importantes](/help/blueprints/business-objectives/overview.md), ajudando você a alinhar o trabalho de implementação aos resultados estratégicos.

```
Industry Use Case → Use Case Pattern → Key Business Objective
```
